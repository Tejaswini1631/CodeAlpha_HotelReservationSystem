# Hotel Reservation System (Java + MySQL)

## Description

This project is developed as part of CodeAlpha Internship.

* Book Room
* Cancel Booking

* MySQL

## How to Run

1. Setup MySQL database
2. Create table using SQL queries
3. Run MainApp.java
